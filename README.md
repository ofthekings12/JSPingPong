# JSPingPong

## A simple, two-player, classic Pong game purely built with HTML, CSS, and Javascript.

Play the game here:

[mikko-jspong.netlify.app](mikko-jspong.netlify.app)

Controls:
Player 1:
- Up = 'W'
- Down = 'S'

Player 2:
- Up = ↑
- Down = ↓
